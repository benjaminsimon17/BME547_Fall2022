# Patient Health Database

Author: Benjamin Simon

## Purpose
The code in this repository will run a patient
health database.
The code is written in Python.

## Specifications
The database will contain:
* Patient Name
    + First Name
    + Last Name
* Age
* Patient ID
* Test Results
        * Test name
        * Test REsult
    1. HDL
    2. LDL
